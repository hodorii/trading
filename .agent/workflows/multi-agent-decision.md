# multi-agent-decision

## 준수: [PROTOCOLS.md] (GP-ETO, GP-EBR)
- **중복 금지**: IDR 팩트 재진술 시 즉시 중단. 오직 **[판단]**과 **[행동]**만 기록.

---

## Pipeline

### 1. Context Sync
- NQ=F, ES=F, 환율 최종 실측 ➡️ 시장 기조(Sentiment) 확정.

### 2. Critical Debate
- **Conflict Resolution**: IDR [합성]의 모순점(예: 저평가이나 이탈) 최종 판결.
- **Scenario**: [If-Then] 설계 (예: 나스닥 -N% 변동 시 대응 수칙).

### 3. Final Commitment
- **[결정]**: 타점 및 실행 시나리오 (전문가용).
- **[가이드]**: 3초 행동 지침 (Checklist 중심).

### 4. Closing
- **[통합]**: 배경 논리 및 세션 종료 보고.
